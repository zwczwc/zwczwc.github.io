Django 安装版本

Django 默认安装最新版本：

pip install django

Django 后面接版本号就可以了：

pip install django==1.11.7

如果使用 pip install 安装库比较慢，可以用豆瓣的镜像：

pip install django==1.11.7 -i http://pypi.douban.com/simple –trusted-host pypi.douban.com

用 pip 安装库报错：Microsoft visual c++ 14.0 is required
